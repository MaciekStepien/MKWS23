## About

This repository contains a computational study of detonation phenomena focusing on Chapman-Jouguet detonation waves in methane-air and hydrogen-air mixtures. The project utilizes numerical simulations and combustion modeling techniques to analyze critical detonation parameters, flame propagation, and shock wave interactions relevant to combustion science and propulsion engineering.

---

## Repository Structure

This is a Python project titled: *Parameters of Chapman-Jouguet Detonation of Methane-Air and Hydrogen-Air Mixtures*.

- `MKWS_Code.py` — Python script implementing numerical simulations of detonation parameters.
- `Report_Maciej_Stepien.pdf` — PDF report detailing methodology, results, and analysis.

> **Note:** The PDF report file may not display correctly in GitHub's file preview. To view the full formatted report, please download it and open locally using a PDF reader.
